# Subtomogram segmentor
Subtomogram segmentor is set of deep learning based architectures for semantic segmentation of Cellular Electron Cryo Tomography subtomogram data

Please refer to our paper for more details:

Chang Liu, Xiangrui Zeng, Ruogu Lin, Xiaodan Liang, Zachary Freyberg, Eric Xing, and Min Xu.  Deep learning based supervised semantic segmentation of electron cryo-subtomograms. In 2018  IEEE  International  Conference  on  Image  Processing,  ICIP, Athens,  Greece, October 7-10, 2018, pages 1578â€“1582. 

https://arxiv.org/abs/1802.04087
